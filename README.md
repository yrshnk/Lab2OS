Проект представляет собой консольное многопоточное приложение на C++, демонстрирующее создание и координацию потоков средствами WinAPI. Программа состоит из трёх потоков: main, min_max и average, совместно обрабатывающих массив целых чисел.

Главный поток (main) считывает с консоли размер массива и его элементы, выделяет динамическую память и запускает потоки min_max и average, передавая им размер массива. После запуска main ожидает завершения обоих потоков с помощью WaitForSingleObject(INFINITE), затем заменяет минимальный и максимальный элементы массива на среднее арифметическое и выводит итоговый массив на консоль.

Поток min_max последовательно просматривает массив, находит минимальный и максимальный элементы, после каждого сравнения выполняя задержку Sleep(7). Найденные значения сохраняются в общих переменных и выводятся на консоль.

Поток average вычисляет среднее арифметическое элементов массива, выполняя Sleep(12) после каждой операции суммирования. Результат сохраняется в общей переменной и выводится на консоль.

Реализация выполнена на стандарте C++17 и использует WinAPI для работы с потоками и синхронизации (CreateThread, WaitForSingleObject, Sleep, CloseHandle). Данные между потоками разделяются через общую память. Проект включает модульные и интеграционные тесты на базе GoogleTest, проверяющие корректность вычислений и модификации массива. Обработка ошибок охватывает контроль ввода данных, успешность создания потоков и корректное освобождение ресурсов.
